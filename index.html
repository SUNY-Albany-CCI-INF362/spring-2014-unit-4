Two templates but same nav & footer?

Use _includes !!

├── _includes
  ├── navigation.html
  └── footer.html

<!doctype html>
<html lang="en">
<body class="one">
    {% include navigation.html %}
    {{ content }}
    {% include footer.html %}
</body>
</html>
<!doctype html>
<html lang="en">
<body class="two">
    {% include navigation.html %}
    {{ content }}
    {% include footer.html %}
</body>
</html>

One last thing...

---
layout: default
title: Page Title
---

that's not all you can use

Custom Template attributes

---
layout: default
title: Page Title
vendorScripts: ['script1.js', 'script2.js']
customScripts: ['easy-tap.js']
customStyles: ['easy-tap.css']
---

Iterate over them

{% for styles in page.customStyles %}
  <link rel="stylesheet" href="css/{{ styles }}.css">
{% endfor %}
{% for scripts in page.vendorScripts %}
  <script src="js/{{ styles }}">
{% endfor %}
